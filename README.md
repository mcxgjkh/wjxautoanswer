# 问卷星自动答题器 V7.0.1
Design by MCXGJKH

基于Electron开发的问卷星自动答题桌面应用程序，新增题库管理和正确率控制功能。

## 版本更新 V7.0.1
✅ **题库管理** - 支持添加、编辑、删除题库，最多1000条题目  
✅ **正确率控制** - 0-100%可调，采用去一法保证实际正确率≥设定值  
✅ **问卷链接自定义** - 可自定义问卷链接后缀  
✅ **默认题库** - 内置V5.0版题库作为默认题库  
✅ **去一法计算** - 保证实际正确率大于等于设定正确率  

## 功能特点

### 1. 题库管理功能
- 支持创建多个题库，每个题库独立管理
- 题目答案支持自动格式化（自动添加字母选项）
- 题库容量自动扩展：每100行自动扩展10行，最多1000行
- 内置默认题库："2026理化生实验操作模拟考试-化学"
- 题库预览功能，可查看题目统计信息

### 2. 正确率控制
- 0-100%无级调节
- 采用去一法计算：需要正确题数 = ceil(有效题目数 × 正确率%)
- 有效题目数 = 总题目数 - 基础信息填写题数
- 保证实际正确率 ≥ 设定正确率

### 3. 问卷链接自定义
- 支持自定义问卷链接后缀
- 基础URL固定为：https://ks.wjx.com/vm/
- 只需填写后缀如：wRSroLM.aspx

### 4. 速度选择
- 9种答题速度可选
- 最快5秒完成所有题目
- 支持随机速度模拟真人操作

## 安装使用

### 方法一：下载预编译版本
1. 下载最新版本的安装包
2. 运行安装程序
3. 启动应用程序

### 方法二：从源码构建
```bash
# 1. 安装Node.js (>=16)
# 2. 克隆或下载本项目
# 3. 安装依赖
npm install

# 4. 运行开发版本
npm start

# 5. 打包成可执行程序
npm run build:win  # Windows 64位
npm run build:win32  # Windows 32位
npm run build:mac  # macOS
npm run build:linux # Linux
