# 问卷星自动答题器 V7.2.3 标准版使用说明
Design by MCXGJKH

基于Electron开发的问卷星自动答题桌面应用程序，新增题库管理和正确率控制功能。

## 版本更新 V7.2.3
✅ **题库管理** - 支持添加、编辑、删除题库，最多1000条题目  
✅ **正确率控制** - 0-100%可调，采用去一法保证实际正确率≥设定值  
✅ **问卷链接自定义** - 可自定义问卷链接后缀  
✅ **默认题库** - 内置V5.0版题库作为默认题库  
✅ **导出导入** - 支持导入、导出题库，json/csv格式 
✅ **全新UI** - 更美观的设计  

## 功能特点

### 1. 题库管理功能
- 支持创建多个题库，每个题库独立管理
- 题目答案支持自动格式化（自动添加字母选项）
- 题库容量自动扩展：每100行自动扩展10行，最多1000行
- 内置默认题库："2026理化生实验操作模拟考试-化学"
- 题库预览功能，可查看题目统计信息

### 2. 正确率控制
- 0-100%无级调节
- 采用去一法计算：需要正确题数 = ceil(有效题目数 × 正确率%)
- 有效题目数 = 总题目数 - 基础信息填写题数
- 保证实际正确率 ≥ 设定正确率

### 3. 问卷链接自定义
- 支持自定义问卷链接后缀
- 基础URL固定为：https://ks.wjx.com/vm/
- 只需填写后缀如：wRSroLM.aspx

### 4. 速度选择
- 9种答题速度可选
- 最快5秒完成所有题目
- 支持随机速度模拟真人操作

# 问卷星自动答题器 V7.2.3 便携版使用说明

## 什么是便携版？

便携版（绿色版）是指无需安装即可使用的软件版本，具有以下特点：

### 优点：
1. **无需安装**：解压即可使用，不修改系统注册表
2. **数据便携**：所有数据保存在程序目录的 `userdata` 文件夹中
3. **方便携带**：可以放在U盘或移动硬盘中，随时随地使用
4. **多版本共存**：可以同时运行多个不同版本
5. **干净卸载**：删除整个文件夹即可完全卸载，不留痕迹

### 文件结构：
问卷星自动答题器 V7.2.3-便携版.exe # 主程序
userdata/ # 数据目录（首次运行自动创建）
├── config.json # 程序配置
├── banks/ # 题库数据
├── logs/ # 日志文件
└── data/ # 其他数据

## 使用说明

### 首次运行：
1. 解压下载的压缩包到任意目录
2. 双击运行 `问卷星自动答题器 V7.2.3-便携版.exe`
3. 程序会自动创建 `userdata` 文件夹用于保存数据

### 数据备份：
1. 要备份所有数据，只需备份整个程序目录
2. 要迁移到新电脑，复制整个文件夹即可

### 注意事项：
1. 请勿删除 `userdata` 文件夹，否则会丢失所有数据
2. 建议将程序放在非系统盘，方便管理
3. 如果遇到权限问题，请以管理员身份运行
